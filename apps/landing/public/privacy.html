2. Welche Daten werden verarbeitet?

Real Life Stack erhebt keine personenbezogenen Daten ohne dein Zutun.

  • Lokales Backend: Alle Daten werden ausschließlich lokal auf deinem Gerät gespeichert (IndexedDB). Es findet keine Übertragung an externe Server statt.
  • GraphQL-Backend (optional): Wenn du dich mit einem eigenen Server verbindest, gelten die Datenschutzbestimmungen des jeweiligen Betreibers.
  • Web-of-Trust-Backend (optional): Daten werden dezentral und Ende-zu-Ende-verschlüsselt synchronisiert.

3. Keine Tracking- oder Analyse-Dienste

Die App verwendet keine Analyse- oder Tracking-Dienste. Es werden keine Nutzungsdaten an Dritte übermittelt.

4. Berechtigungen

Die App fragt nur die Berechtigungen an, die für die genutzten Funktionen notwendig sind (z. B. Standort für die Kartenansicht). Es werden keine Daten ohne Einwilligung erhoben.
